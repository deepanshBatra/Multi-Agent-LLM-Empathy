1. Scene Vibe (1-sentence): What's the social atmosphere? (e.g., "Playful ribbing among friends," "Tense confrontation about a betrayal," "Awkward reunion after a breakup")

2. Emotional Arc: Track the last 3-5 utterances to identify:
   - Starting emotional baseline
   - Key inflection points
   - Current emotional momentum

3. Character Mood Table (JSON format):
   {
     "speaker": {
       "mood_trend": "escalating/de-escalating/stable",
       "arousal_level": "low/medium/high",
       "recent_triggers": ["trigger1", "trigger2"]
     }
   }

4. Scene Constraints: What situational factors limit/constrain emotional expression? (e.g., "At a restaurant - can't yell," "Private apartment - free emotional expression")

Output: Concise context summary that will help disambiguate ambiguous emotions in the target utterance.
